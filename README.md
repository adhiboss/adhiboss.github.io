## ✨ Features
- **Interactive UI**: Engaging user interface with a background generated using Three.js and custom CSS animations.
- **Multilingual Support**: Logo and key phrases that smoothly switch between English and Kannada text.
- **Terminal Theme Projects**: Projects showcased in interactive, terminal-styled cards.
- **Timeline Sections**: Visually appealing timelines for both my educational journey and extracurricular achievements.
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Libraries**: 
  - [Three.js](https://threejs.org/) (for 3D background elements)
  - [FontAwesome](https://fontawesome.com/) (for icons)
  - [Devicons](https://devicon.dev/) (for tech stack icons)
- **Fonts**: JetBrains Mono, Orbitron, Space Grotesk (Google Fonts)
- **Forms**: [Formspree](https://formspree.io/) (Handling contact form submissions)
